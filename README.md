# Fitness Tracker

This a simple Fitness Tracker website that allows users to input their workout and walking routines, to view various dish recipes (Need to populate the database) and save them as favorites, and view some of their statistics.
This website is part of a group project for a university course. 

The project doesn't implement very good security / error handling features properly as it was not done to be production ready and merely for display purposes "It was also done solo and quickly so didn't have time to implement 
the appropiate measures". 
Issues can be found mostly in the API routes for POST handling where there is no proper sanitization for user inputted data with a naive JWT approach as the sole middleware implemented.
Not particulry proud of this project but it still exists so it shall remain.

## Recommended IDE Setup

Vue 3 defaults.
[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur).

## Project Setup

```sh
npm install
```

## Project Run

### Run Server

```sh
node express.js
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```
